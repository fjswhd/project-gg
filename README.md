# 같이가치

<hr>

## 개발 목표
<p>Spring framework 와 카카오맵 API를 활용해 함께할 메이트를 찾아주는 웹 페이지를 구축</p>
<hr>

## 사용 기술
<ul>
	<li>Java</li>
	<li>Spring Framework</li>
	<li>Jsp</li>
	<li>JavaScript (jQuery, Ajax, kakaoMap API)</li>
	<li>Bootstrap</li>
	<li>CSS</li>
</ul>
